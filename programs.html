Programs

The Foundations Program

A guided, project-based path into web technology — built for complete beginners, and built around momentum.

Overview

Twelve weeks, one project at a time

The Foundations Program is a twelve-week, mentor-led journey through the fundamentals of web technology. Instead of front-loading theory, each week is anchored to a real project — so students see, immediately, what their new skills are for.

Cohorts are kept small so mentors can give real attention to every student. Sessions mix short, focused instruction with long stretches of guided build-time, pair programming, and code review.

Curriculum highlights

What the twelve weeks look like

Weeks 1–3 · Foundations of the web

HTML structure, CSS styling, and the basics of responsive, accessible layout. Ends with a personal landing page.

Weeks 4–7 · Bringing pages to life

JavaScript fundamentals — variables, logic, events, and DOM manipulation. Ends with an interactive browser game.

Weeks 8–10 · Thinking like a developer

Debugging, version control, and working with real-world tools. Students collaborate in pairs on a shared project.

Weeks 11–12 · Capstone & showcase

Students plan, build, and present an original capstone project to mentors, peers, and family at Demo Day.

Who can apply

Built for beginners, open to anyone

  • Ages 16–24, no prior coding experience required
  • Access to a computer and internet for program hours
  • Curiosity, and a willingness to show up and try
  • No cost to apply or attend — ever
