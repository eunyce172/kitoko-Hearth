Landing Pages

The very first project every Foundations student ships: a fully responsive, hand-coded landing page for a cause, idea, or small business of their choosing. Students practice semantic HTML, layout with CSS Grid and Flexbox, and thinking about a real visitor's experience.

HTML5 CSS3 Responsive Design

Memory Match Game

An interactive, fully playable memory-matching game built entirely in JavaScript. Students implement game state, card-flip logic, a scoring system, and delightful little win animations — their first taste of building something truly interactive.

JavaScript DOM Events Game Logic

Landing Pages

Students design and build a fully responsive landing page from a blank file — no starter templates. The project covers accessible semantic structure, a mobile-first layout system, and enough CSS polish to make it feel like a real, launch-ready page.

Skills practiced: semantic HTML5, CSS Grid & Flexbox, responsive breakpoints, accessible color contrast and focus states.

🧠

Memory Match Game

A browser-based memory matching game, fully playable start to finish. Students build the shuffle logic, track flipped and matched cards in state, keep score, and add a small celebratory animation when the board is cleared.

Skills practiced: JavaScript fundamentals, DOM manipulation, event handling, basic game-state management.

📊

Personal Dashboards

A capstone project where students connect a public API to a custom-built interface — tracking something personal, from a weather widget to a simple habit tracker — and present it at Demo Day.

Skills practiced: working with APIs, asynchronous JavaScript, data-driven UI, presenting technical work.
